Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The meta element is a container for metadata.
  • The image element is used to embed an image.
  • The source element specifies the path to the image.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Do Not Repeat Yourself (DRY) is a principle that encourages developers to reuse code to reduce the number of lines of code.
  • A text-align is a way to center a title or heading on a webpage.
  • A display property is assigned with the block value to assign each image to start on a new line.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • gi branch: lists your branches
  • git merge: merges the specified branch's history into the current branch
  • git log: shows all the commints in the current branch's history
  • git add -A: this commits all files in the working branch
